## 📖 项目简介 | Introduction

**English**  
*poker-abc* is a set of **multilingual language‑pair flashcards** designed in the style of **early‑learning playing cards**. Each card includes **trace‑writing practice** and **QR‑code audio pronunciation**, making it suitable for learners of all ages.  
Printable files are **free for personal use**, while **commercial use requires a paid license**.

**中文**  
*poker-abc* 是一套 **多国语言语言对启蒙闪卡**，采用 **启蒙扑克牌式设计**，每张卡片包含 **描红练习** 与 **扫码读音** 功能，适合所有年龄段的语言学习者。  
可打印文件 **个人免费使用**，**商业使用需付费授权**。
